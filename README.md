# DOM Manipulation

## Overview
This week focuses on consolidating foundational skills in **HTML, CSS, and JavaScript**, while beginning to integrate **backend concepts with Express**. The goal is to polish existing portfolio components and prepare for modular backend integration.

---

## Learning Objectives
- Strengthen understanding of **semantic HTML** and accessibility.
- Apply **responsive design** using CSS Grid and Flexbox.
- Build and refine **reusable components** (cards, galleries, forms).
- Practice **JavaScript fundamentals** with interactive mini-projects.
- Scaffold a **modular Express backend** with routes and middleware.

---

## Tasks Completed
- Converted portfolio sections into **semantic HTML**.
- Implemented **responsive grid layouts** for galleries.
- Created **hero sections** with full viewport height and overlays.
- Integrated **contact forms** and blog post templates.
- Scaffolded **React components** for practice exercises.
- Set up **Express backend structure** with routes, controllers, and environment variables.

---

## Key Learnings
- Importance of **semantic markup** for accessibility.
- How **CSS Grid and Flexbox** simplify responsive layouts.
- Benefits of **modular backend design** for scalability.
- Debugging workflow with **VS Code + PowerShell**.

---

## Summary
Week 5 was about **solidifying frontend foundations** and **laying the groundwork for backend integration**. The portfolio is now structured, responsive, and ready for dynamic features in the coming weeks.
